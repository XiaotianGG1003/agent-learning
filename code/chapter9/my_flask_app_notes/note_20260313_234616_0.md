# 项目架构分析

项目采用扁平化结构，包含5个核心模块：
1. api_client.py - API交互
2. data_processor.py - 数据处理
3. models.py - 数据模型
4. utils.py - 工具函数
5. __init__.py - 包初始化

无子目录结构，适用于小型项目。
